### PROJECT NAME: Attention-Focused Eye Gaze Analysis to Predict Autistic Traits Using Transfer Learning, GenAI, XAI and ViT
• Description: This project uses eye gaze analysis to predict autistic traits by leveraging Vision Transformers and Transfer Learning. Generative AI augments data for better performance, and Explainable AI provides
insights into which gaze patterns are most predictive. The approach offers an accurate and interpretable framework for early detection of autism spectrum disorder.
• Tools Used: Python, NumPy, Pandas, Matplotlib, Seaborn, Plotly, Scikit-learn (Sk-learn),
PyTorch, OpenCV, Transfer Learning, GenAI, XAI, ViT, Jupyter Notebook
